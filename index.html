CSS & HTML Cheatsheet

HTML Table Cheat Sheet

Tag

Name

Description
<table> Table Creates a table
<th> Table Head Defines a table head
<tr> Table Row Defines a table row
<td> Table Data Defines a table cell
<caption> Table Caption Defines a table caption
<colgroup> Column Group Specifies a group of one or more columns in a table for formatting
<col> Column Specifies column properties for each column within a <colgroup> element
<thead> Table Header Groups table header content
<tbody> Table Body Groups table body content
<tfoot> Table Footer Groups table footer content

Attribute

Name

Description
colspan Column Span Make a column span over multiple columns
rowspan Row Span Make a row span over multiple row

CSS Box Model

Name

Modifiers

Description
Content width, height Box where content sits
Padding padding(-x) Space between content and border
Border border-style(-x), border-color, border-width, border-radius Border around content + padding, shorthand: border: width style color
Margin margin(-x) Space between border and everything else
